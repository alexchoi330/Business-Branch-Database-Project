Steps to run this project:
1. Download the github files
2. Move all the php files into the public_html folder on remote.students.cs.ubc.ca
3. Make sure all files have the right permissions (chmod 711 ~ and chmod 755 ~)
4. Go to https://www.students.cs.ubc.ca/~YOURCWLHERE/project.php
5. Press the reset button once
6. Press the generate tables button
OR 
Run the project through this link:
https://www.students.cs.ubc.ca/~yuxuanhe/project.php




Note: The .sql script to run the file exists and works, but the functionality is also implemented in the project.php file for convenience through the reset and generate table buttons.
